# vocab-universe
# 🌟 vocab-universe – 5 词族沉浸式单词学习（进化中）

> 不是背单词，是看见单词之间的星系联系。

---

## 📱 产品定位

vocab-universe 是一个**词族驱动的单词学习工具**，以 5 个核心词族为起点，通过星图可视化、场景词群、易混辨析等模块，帮助用户建立**语义网络**，而不是孤立背单词。

**目标用户**：备考党（雅思/托福）、英语爱好者、想要系统扩充词汇量的学习者。

---

## 🧩 四个核心模块（4 标签页）

| 标签页 | 功能 | 状态 |
| :--- | :--- | :--- |
| 🌌 **星图** | 2.5D 交互式词族星系，点击单词卡片查看释义、例句、多义项 | ✅ 已完成 |
| 🌳 **词族** | 纵向折叠树，展示 5 个词族的层级结构（词根 → 派生词 → 短语） | ✅ 已完成 |
| 🏞️ **场景** | 横向滑动卡片，展示 5 词族在真实语境（学术/职场/生活）中的用法 | ✅ 已完成 |
| 🔍 **辨析** | 对比卡片，区分易混近义词（如 dictate / predict，affect / effect） | ✅ 已完成 |

> 激励体系（连续打卡、进度条）和 FSRS 复习调度因需要后端存储，当前版本为静态展示，后续迭代。
